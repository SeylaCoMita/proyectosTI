APUNTES DE HTML

¿Qué es HTML?

HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el lenguaje estándar para crear páginas web y estructurar su contenido mediante etiquetas como <p>, <img>, <a>, entre otras.

Elementos de Bloque

Ocupen todo el ancho disponible y comienzan en una nueva línea:

  • <p>: párrafos
  • <div>: divisiones o contenedores
  • <h1> - <h6>: encabezados del más grande al más pequeño
  • <ol>: lista ordenada
  • <ul>: lista desordenada
  • <form>: formulario de entrada de datos
  • <header>, <nav>, <article>, <aside>, <footer>: elementos semánticos de HTML5

Elementos de Línea

Solo ocupan el ancho necesario y no generan saltos de línea:

  • <small>: texto con menos énfasis
  • <wbr>: permite salto de línea si es necesario
  • <time datetime="">: representa una fecha u hora
  • <span>: para aplicar estilos en línea
  • <a>: enlaces
  • <img>: insertar imagen
  • <strong> y <em>: enfatizan texto (negrita y cursiva)
  • <b>, <i>, <u>: estilo visual (negrita, cursiva, subrayado)
  • <br>: salto de línea

Ejemplo: <span> es un elemento de línea.
